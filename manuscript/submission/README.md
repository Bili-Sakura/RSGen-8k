# ICPR 2026 Submission

Manuscript template for ICPR 2026, based on Springer LNCS format.

## Files

- `main.tex` - Main manuscript (placeholder structure, no concrete content)
- `references.bib` - Bibliography (BibTeX)
- `llncs.cls` - LNCS document class
- `splncs04.bst` - Bibliography style
- `fig1.eps` - Sample figure (replace with your own)

## Build

```bash
pdflatex main
bibtex main   # if using .bib
pdflatex main
pdflatex main
```

Or use your preferred LaTeX editor/compiler.
